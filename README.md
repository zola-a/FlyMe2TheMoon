# SQL + Visual Studio Projects

## 📌 Overview
This repository contains database and application projects built using **SQL Server** and **Visual Studio (VB.NET)**. The projects demonstrate how desktop applications interact with relational databases using SQL queries, stored procedures, and role-based system design.

---

## 🧩 Main Project: FlyMe2TheMoon

A role-based flight management system built using **VB.NET (Visual Studio)** and **SQL Server**.

The system supports two main user roles: **Passenger** and **Employee/Admin**, each with different permissions and functionality.

---

## ✈️ Core Features

### 👤 Passenger Side
- Secure login with credentials
- Book flights
- View upcoming and scheduled flights

### 🛠️ Employee/Admin Side
- Secure admin login system
- Manage flight operations (create, assign, delete flights)
- Assign pilots and manage schedules
- Track employee attendance
- Manage aircraft assignments
- View and monitor aircraft maintenance status

---

## 🛠️ System Highlights
- Role-based authentication system (Passenger vs Employee)
- SQL Server database integration
- Stored procedures for secure database operations
- Full CRUD functionality (Create, Read, Update, Delete)
- Desktop application built with VB.NET
- Real-world airline management workflow simulation

---

## 🗄️ Database Features
- Relational database design
- Flight, passenger, and employee tables
- Stored procedures for data handling
- Structured data integrity and relationships

---

## 🛠️ Technologies Used
- Visual Studio
- VB.NET
- SQL Server
- T-SQL (Tables, Queries, Stored Procedures)

---

## 🚀 How to Run the Project
1. Open SQL Server and run all `.sql` scripts in the Database folder  
2. Open the `.sln` file in Visual Studio  
3. Build the solution  
4. Run the application  
5. Ensure database connection string is properly configured
